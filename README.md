# freeCodeCamp-zod-to-openapi-scalar

A project demo for the freeCodeCamp article on the same project name.

## Requirements

- Node.js version >= `22`
- npm

## Setup

After cloning the project, install the project's dependencies using your Node.js package manager.

Run the project with `npm run serve` and you can access the endpoints on [http://localhost:3000](http://localhost:3000)
